<p>In the ever-evolving landscape of modern application development, observability and operational control are no longer optional – they’re essential. At Hasura, we understand platform owners' and developers' challenges in managing complex, data-driven environments. That’s why we’re thrilled to introduce the <a href="https://hasura.io/docs/3.0/observability/platform-dashboard/"><strong>Platform Dashboard</strong></a> for Hasura Data Delivery Network (DDN) – a comprehensive solution to elevate your operational efficiency, data observability, and decision-making.</p><p>The Platform Dashboard is now available to all Hasura DDN users on <a href="https://hasura.io/pricing"><strong>Base plans and up</strong></a>. Whether managing a single project or multiple interconnected initiatives, this feature provides the visibility and control you need to optimize your API operations. Explore the Platform Dashboard in your Hasura DDN projects today and experience the future of observability.</p><h2 id="what-is-the-platform-dashboard">What is the Platform Dashboard?</h2><p>The Platform Dashboard is a centralized interface designed to give platform owners and developers critical insights into their Hasura DDN operations. By providing detailed metrics, visualizations, and actionable analytics, the dashboard empowers you to:</p><ul><li><strong>Understand usage patterns:</strong> Gain insights into which APIs are used most frequently, who is accessing your data, and how often.</li><li><strong>Ensure compliance and security:</strong> Monitor data access patterns to identify potential anomalies and ensure adherence to regulatory requirements.</li><li><strong>Optimize performance:</strong> Track API latency, request throughput, and error rates to pinpoint areas for improvement.</li><li><strong>Streamline operations:</strong> Manage multiple projects seamlessly with unified visibility across all your APIs and data sources.</li></ul><h2 id="the-four-pillars-of-the-platform-dashboard">The four pillars of the Platform Dashboard</h2><p>The Platform Dashboard is organized into four key sections, each designed to address critical aspects of observability and control:</p><h3 id="1-security-and-governance">1. <strong>Security and governance</strong></h3><figure class="kg-card kg-image-card"><img alt="" class="kg-image" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXcFI6k7HixZm5UB7ebwB81Mdb1XsvbtQaGlyKjxL5x5N8avgSdxYGiWTLIaQfPdQ85yEzuzM0tCKjUORGiaVNChq4NSGUOUHKa2a6VMEgPQSr2VUny2ikvGvRkmsjMbsf8SsCsd?key=g0DM84egrtVgMAe01KGwIb8J" /></figure><p>Ensuring the security and compliance of your data infrastructure is paramount. The security and governance section provides:</p><ul><li><strong>Authentication status:</strong> View and resolve authentication issues, including configuration status and types (e.g., JWT).</li><li><strong>Permission analysis:</strong> Monitor model- and field-level security, identifying unrestricted access and recommending tightening controls.</li><li><strong>Documentation coverage:</strong> Assess and improve API documentation, with metrics on description coverage for models, fields, and commands.</li></ul><h3 id="2-api-reliability">2. <strong>API reliability</strong></h3><figure class="kg-card kg-image-card"><img alt="" class="kg-image" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXeNEBRAbOco0Jf1oWK4xXcp4SVaDAScdnlQV2h78Q3UG_YnZ3SjHCcIXAPiu1V_zaGBhB4nrDH-soBk2qtyHe1DaGAcSWN3j34eGZq5euV0dnQRwigNZCQbamm2o6ljmrdGcFRtJg?key=g0DM84egrtVgMAe01KGwIb8J" /></figure><p>Reliability is the backbone of user trust. The API reliability section enables you to:</p><ul><li>Monitor <strong>system reliability</strong> with metrics like error-to-success ratios.</li><li>Identify patterns in API errors and successes to ensure a consistently high-quality user experience.</li></ul><h3 id="3-api-usage-trends">3. <strong>API usage trends</strong></h3><figure class="kg-card kg-image-card"><img alt="" class="kg-image" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXcTJU_22K1_6_1DU1vvL29ceV_PrWFhj24MFx8iGnnT2SzSs8Dg3nfwhUcYYitFIJhjxVpB6ozQ-A_pgvy3hSVgc1SzOwEpNFmNRJ8FwqgYlAZfXSl8xIc0p0aTmhRF-uIZpQxi9Q?key=g0DM84egrtVgMAe01KGwIb8J" /></figure><p>Understanding how your APIs are used is key to optimizing performance and scaling effectively. The API usage trends section offers:</p><ul><li>Metrics on <strong>requests per day</strong> to track API consumption over time.</li><li>Insights into deprecated metadata and features, helping you maintain a clean and efficient API.</li></ul><h3 id="4-developer-productivity">4. <strong>Developer productivity</strong></h3><figure class="kg-card kg-image-card"><img alt="" class="kg-image" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXeOSGOeqi7FfR6JwTkcVukOoNin0xR5tPxpQNGdrKI04PdDBksJYqaSwMDVvnnhnzTSB8o8WJwjsurGpdT5820gF_y3J2a-ciJwT9koS5vnB-A1crUtVpie6ntoQzTVabiJeemh?key=g0DM84egrtVgMAe01KGwIb8J" /></figure><p>Fostering collaboration and efficiency across teams is critical for scaling your operations. The developer productivity section provides:</p><ul><li><strong>Project access distribution:</strong> A breakdown of user roles (Owner, Admin, Read-only) and total user counts.</li><li><strong>Subgraph access metrics</strong> (Advanced plans)<strong>:</strong> Collaboration insights per subgraph, including admin and read-only user distributions.</li></ul><p></p><h2 id="how-to-get-started">How to get started</h2><p>The Platform Dashboard is available to all Hasura DDN users on Base plans and up. To explore its capabilities:</p><ol><li>Log in to your Hasura DDN console.</li><li>Navigate to the **Platform Dashboard** section.</li><li>Start monitoring your API performance, data access patterns, and project metrics.</li></ol><p>For a detailed walkthrough, check out the <a href="https://hasura.io/docs/3.0/observability/platform-dashboard">Platform Dashboard documentation</a>.</p><h2 id="join-the-conversation">Join the conversation</h2><p>We’d love to hear your thoughts on the Platform Dashboard. Try it out in your Hasura DDN projects and share your feedback.
